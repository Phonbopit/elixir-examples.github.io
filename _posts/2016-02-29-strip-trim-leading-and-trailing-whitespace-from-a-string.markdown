Remove/trim leading and trailing whitespace from a string

{% highlight elixir %}
"a" = String.strip("    a     ")
{% endhighlight %}

Documentation: [String.strip/1](http://elixir-lang.org/docs/stable/elixir/String.html#strip/1)
